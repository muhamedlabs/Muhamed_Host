## About the Project

Muhamed_Host offers **affordable hosting** for a wide range of tasks:

- **Discord and Telegram bots** – from $1/month.
- **Game servers** – Minecraft (Vanilla, Forge, Paper, Spigot, FTB), Rust, ARK, CS, Satisfactory, Conan Exiles, and more.
- **Voice servers** – Team Speak 3 with mod support (JTS3ServerMod).
- **Additional services** – databases, web applications (coming soon), scripts in Python, Node.js, Go, etc.

> **Important**: currently **server provisioning is done manually** – this is a temporary approach that allows us to keep prices low. Full automation via the Pterodactyl API is planned for the future.

## Ordering Process

Currently, there is no automation due to economic reasons. The process is as follows:

1. The client sends a request directly via Telegram / Email (providing name, contact, server type, and requirements).
2. We receive the request and contact the client to clarify details.
3. We discuss the configuration and agree on the final price.
4. After payment, we manually create an account in Pterodactyl and set up the server according to the requirements.
5. The client receives access to the control panel and can start using the server.

This approach allows us to accommodate any client preferences and keep prices minimal.
