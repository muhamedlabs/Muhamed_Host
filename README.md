**Muhamed_Host** — це хостинг-платформа, що працює на базі **[Pterodactyl](https://pterodactyl.io/)** з власними покращеннями від **Muhamed IT**. Ми надаємо сервери для Minecraft, Rust, Team Speak, Discord/Telegram ботів та багатьох інших застосунків. Цей репозиторій містить код лендінг-сайту, через який клієнти можуть подавати заявки на створення сервера.

<img src="https://github.com/muhamedlabs/MuhamedTrash/blob/main/Cache/Sticks.png?raw=true">

---

## 📌 Про проєкт

Ми пропонуємо **недорогий хостинг** для найпопулярніших ігор та ботів:

- 🤖 **Боти для Discord і Telegram** – від $1/міс.
- 🎮 **Ігрові сервери** – Minecraft (Vanilla, Forge, Paper, Spigot, FTB), Rust, ARK, CS, Satisfactory, Conan Exiles тощо.
- 🗣️ **Голосові сервери** – Team Speak 3 із модами.
- ⚙️ **Інші застосунки** – бази даних, сервіси на Python, Node.js, Go та інші (веб-хостинг з’явиться згодом).

**Особливість**: наразі **видача серверів відбувається вручну** — це тимчасове рішення, яке дозволяє нам тримати низькі ціни. У майбутньому планується повна автоматизація через API Pterodactyl.

---

## 🛠️ Технології

- **Платформа**: Pterodactyl (з кастомними доповненнями від Muhamed IT)
- **Frontend**: HTML5, CSS3, JavaScript (ванільний)
- **Форма заявки**: надсилання даних через Telegram Webhook або Email (налаштовується)
- **Деплой**: будь-який статичний хостинг (Vercel, Netlify, GitHub Pages тощо)

Повний перелік доступних серверів (яєць Pterodactyl) можна знайти за посиланнями:

Наш хостинг підтримує всі типи, представлені в цих каталогах, зокрема:

- **Minecraft** – будь-які версії та моди.
- **Rust** – з Oxide або без.

- **Discord/Telegram боти** – на будь-якій мові (Python, JS, Java, C# тощо).

## 📞 Процес замовлення (ручна видача)

Наразі автоматизація відсутня через економічні причини. Процес виглядає так:

1. Клієнт заповнює форму на сайті (ім'я, контакт, тип сервера, вимоги).
2. Ви отримуєте заявку в Telegram (або на Email) і зв'язуєтесь із клієнтом.
3. Обговорюєте деталі, узгоджуєте фінальну вартість.
4. Після оплати ви вручну створюєте обліковий запис у Pterodactyl і налаштовуєте сервер згідно з вимогами.
5. Надаєте клієнту доступ – він заходить у панель і користується сервером.

Цей підхід дає змогу гнучко реагувати на будь-які запити та тримати ціни мінімальними.
