## О проекте

Muhamed_Host предлагает **доступный хостинг** для самых разных задач:

- **Боты для Discord и Telegram** – от $1/мес.
- **Игровые серверы** – Minecraft (Vanilla, Forge, Paper, Spigot, FTB), Rust, ARK, CS, Satisfactory, Conan Exiles и другие.
- **Голосовые серверы** – Team Speak 3 с поддержкой модов (JTS3ServerMod).
- **Дополнительные сервисы** – базы данных, веб-приложения (скоро), скрипты на Python, Node.js, Go и т.д.

> **Важно**: сейчас **выдача серверов происходит вручную** – это временный подход, позволяющий нам удерживать минимальные цены. В будущем планируется полная автоматизация через API Pterodactyl.

## Процесс заказа

Сейчас автоматизация отсутствует по экономическим причинам. Процесс выглядит так:

1. Клиент отправляет заявку через форму напрямую в Telegram / Email (указывает имя, контакт, тип сервера и требования).
2. Мы получаем заявку, связываемся с клиентом для уточнения деталей.
3. Обсуждаем конфигурацию, согласовываем финальную стоимость.
4. После оплаты мы вручную создаём учётную запись в Pterodactyl и настраиваем сервер согласно требованиям.
5. Клиент получает доступ к панели управления и может пользоваться сервером.

Такой подход позволяет учесть любые пожелания клиента и держать цены минимальными.
