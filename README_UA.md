## Про проєкт

Muhamed_Host пропонує **доступний хостинг** для найрізноманітніших завдань:

- **Боти для Discord і Telegram** – від $1/міс.
- **Ігрові сервери** – Minecraft (Vanilla, Forge, Paper, Spigot, FTB), Rust, ARK, CS, Satisfactory, Conan Exiles та інші.
- **Голосові сервери** – Team Speak 3 із підтримкою модів (JTS3ServerMod).
- **Додаткові сервіси** – бази даних, веб-застосунки (незабаром), скрипти на Python, Node.js, Go тощо.

> **Важливо**: наразі **видача серверів відбувається вручну** – це тимчасовий підхід, який дозволяє нам утримувати мінімальні ціни. У майбутньому планується повна автоматизація через API Pterodactyl.

## Ціноутворення

- **Базовий тариф** – $1/міс для ботів і невеликих ігрових серверів:
  - 1 vCPU
  - 1 ГБ RAM
  - 10 ГБ SSD
  - Безлімітний трафік

- **Індивідуальний тариф** – договірна ціна, залежно від необхідних ресурсів:
  - Додаткові ядра CPU
  - Збільшений об’єм RAM або диска
  - Спеціальні моди чи налаштування
  - Пріоритетна підтримка

## Процес замовлення

Наразі автоматизація відсутня через економічні причини. Процес виглядає так:

1. Клієнт надсилає заявку через форму напряму в Telegram / Email (вказує ім'я, контакт, тип сервера та вимоги).
2. Ми отримуємо заявку, зв'язуємось із клієнтом для уточнення деталей.
3. Обговорюємо конфігурацію, узгоджуємо фінальну вартість.
4. Після оплати ми вручну створюємо обліковий запис у Pterodactyl і налаштовуємо сервер згідно з вимогами.
5. Клієнт отримує доступ до панелі керування та може користуватися сервером.

Такий підхід дає змогу врахувати будь-які побажання клієнта та тримати ціни мінімальними.
